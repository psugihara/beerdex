AWS SDK for iOS Readme

The AWS SDK for iOS provides a library, code samples, and documentation for developers to build connected mobile applications using AWS.

For more information about using the SDK, please reference the online documentation here: aws.amazon.com/documentation/sdkforios


To learn more about iOS Development

For more information on iOS development, see the Apple Developer Connection web site at: http://developer.apple.com/devcenter/ios/index.action.
